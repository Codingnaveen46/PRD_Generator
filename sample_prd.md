# Product Requirements Document: Smart To-Do List App

## 1. Overview
The Smart To-Do List App helps users manage their daily tasks efficiently by automatically categorizing them using AI.

## 2. Goals
- Increase user productivity by 20%.
- Reduce time spent organizing tasks.

## 3. Features
1. **Task Creation**: Users can add text-based tasks.
2. **AI Categorization**: The system automatically assigns a category (Work, Personal, Errands) based on the task description.
3. **Due Dates**: Users can set optional due dates.

## 4. User Flows
- User opens app -> Clicks "Add Task" -> Types "Buy milk" -> System categorizes as "Errands".
